#### 核心功能
* 🌍 多语言支持: 中文｜英文｜日文｜韩文｜德文
* 🚀 动态图标: 根据 CPU 使用率自动变更 GIF 播放速度, 支持上传自定义图片
* 🌈 炫彩签名: 用渐变色彩表达每日心情状态
* 📋 剪切板: 简洁高效的剪切板管理工具
* 🖥️ 屏幕圆角: 让屏幕变成圆角, 更加切合 macOS 26 的整体风格
* 📎 菜单栏折叠: 用两个简约的原点标记要收起的应用图标, 一键开合, 方便快捷
* 🪫 系统监控: 在状态栏监控系统状态, 可自由配置监控项
